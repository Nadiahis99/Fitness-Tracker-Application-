# Fitness Tracker Application

A Java-based fitness tracking application that helps users manage workouts, diet plans, and track their fitness goals.

## Features

- Track daily workouts and exercises
- Manage personalized diet plans
- Monitor heart rate and activity levels
- Support for different fitness goals (Weight Loss, Muscle Gain)
- Modular and extensible design using **Design Patterns**

## Design Patterns Used

- **Factory Pattern**: For creating different types of exercises and diet plans
- **Singleton Pattern**: To manage user goals and activity tracking globally
- **Decorator Pattern**: To enhance workouts and diet plans dynamically
